<design-system>
DESIGN STYLE: WINDOWS 95 RETRO OS PORTFOLIO

1. DESIGN PHILOSOPHY
This design recreates the Windows 95 desktop experience as a personal portfolio. The interface behaves like a lightweight operating system.

2. COLOR SYSTEM
desktop-bg: #008080
window-bg: #C0C0C0
taskbar-bg: #BDBDBD
titlebar-active: #000080
titlebar-inactive: #808080

3. TYPOGRAPHY
System fonts only (MS Sans Serif, Tahoma, Segoe UI).

4. LAYOUT
Boot sequence, desktop, draggable windows, taskbar.

5. COMPONENTS
Beveled windows, square buttons, desktop icons.

6. PORTFOLIO WINDOWS
About.exe, Projects.exe, Skills.exe, Contact.exe

7. MOTION
Fast, mechanical animations.

8. ACCESSIBILITY
Keyboard navigation, ARIA roles, reduced motion support.

9. AUTHENTICITY RULES
No gradients, no rounded corners, no modern skeuomorphism.

10. EMOTIONAL GOAL
Feels like Windows 95, but better.
